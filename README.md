## 📌 Program 1: Palindrome Checker

<img width="975" height="875" alt="image" src="https://github.com/user-attachments/assets/cc3da560-8510-4c79-a8d2-ad2fdec953d3" />


OnlineGDB Link : https://onlinegdb.com/D23jAsNk-

### 🔹 Description

This program checks whether a given string is a palindrome or not.

A palindrome is a string that reads the same forward and backward.
Example: madam, racecar

### 🔹 How it works

* Takes input string from user
* Calculates the length manually
* Compares characters from start and end
* If all characters match → Palindrome

### 🔹 Example

Input: madam
Output: The given string is a Palindrome.

Input: hello
Output: The given string is NOT a Palindrome.

## 📌 Program 2: Character Frequency Counter

<img width="1016" height="933" alt="Screenshot 2026-03-22 074617" src="https://github.com/user-attachments/assets/1e62af00-15e3-475c-b9d2-03fdd69e601d" />

### 🔹 Description

This program counts how many times each character appears in a string.

### 🔹 How it works

* Takes input string
* Iterates through each character
* Counts duplicates
* Marks counted characters as '0' to avoid repetition

### 🔹 Example

Input: apple

Output:
a = 1
p = 2
l = 1
e = 1
